# Currency Converter Application

A data-driven Python GUI application that converts currencies using real-time exchange rates from the Free Currency API.

This project was created as part of a school assessment.


## Features

- Convert between multiple currencies using live exchange rates
- Display current exchange rates
- View historical exchange rate data
- GUI built with Tkinter
- API integration using Free Currency API


## Technologies Used

- Python 3
- Tkinter (GUI)
- Requests (API calls)
- Free Currency API
